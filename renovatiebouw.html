Full-stack - Renovatiebouw

  • Vak: Web & Mobile Full-stack
  • Periode: 2de semester 2021
  • Technologie: Laravel, Vue.js, MySql
  • Score: 18/20
Github east

Description

Voor het vak Web & Mobile Full-stack kregen wij als opdracht om voor een renovatiebedrijf een tool te ontwikkelen die het mogelijk maakt om de volledige flow van opmaken van offerte tot oplevering te beheren.

De applicatie is ontwikkeld in een dockeromgeving (gekregen boilerplate). Voor de back-end is er een API ontwikkeld in het framework Laravel. De front-end, gemaakt met Vue.js, gebruikt deze API om vervolgens data op te halen en te verzenden naar de MySql database.
